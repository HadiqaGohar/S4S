# AI Prompt Chaining Task

## Step 1 · Action

Real Task Chosen:
Prepare for a Junior Web Developer Interview.

## Step 2 · Action

AI Tool Used:
ChatGPT

## Step 3 · Action

Task broken into multiple sub-prompts.

### Prompt 1 – Identify Key Topics

"I am preparing for a Junior Web Developer interview. List the most important topics I should study, including HTML, CSS, JavaScript, React, Git, APIs, and problem-solving skills."

### Prompt 2 – Generate Interview Questions

"Act as a technical interviewer. Ask me 20 Junior Web Developer interview questions, one at a time, and wait for my answer before moving to the next question."

### Prompt 3 – Mock Interview

"Conduct a mock Junior Web Developer interview. Ask technical and behavioral questions. After each answer, provide feedback and suggest improvements."

### Prompt 4 – Improve Weak Areas

"Based on my interview answers, identify my weak areas and create a 7-day study plan to improve them."

## Step 4 · Action

Ran each prompt in sequence and fed the output from one prompt into the next.

### Prompt Chain Workflow

#### Output 1: Key Topics Identified

* HTML Semantics
* CSS Flexbox & Grid
* Responsive Design
* JavaScript ES6+
* DOM Manipulation
* React Components & Hooks
* REST APIs
* Git & GitHub
* Debugging Techniques
* Basic Data Structures

#### Output 2: Interview Questions Generated

Examples:

* What is the difference between let, const, and var?
* What is the DOM?
* Explain Flexbox.
* What is a React component?
* How does Git help developers?

#### Output 3: Mock Interview Created

* Technical questions with model answers.
* Behavioral questions with expected responses.
* Feedback and improvement suggestions after each answer.

#### Output 4: Personalized Study Plan

Day 1: HTML & Accessibility

Day 2: CSS Layouts

Day 3: JavaScript Fundamentals

Day 4: DOM & Events

Day 5: React Basics

Day 6: APIs & Async JavaScript

Day 7: Mock Interview Practice

## Step 5 · Action

Documented the entire prompt chain and saved the final deliverable.

### Final Deliverable

A complete AI-assisted interview preparation workflow including:

* Important study topics
* Common interview questions
* Mock interview session
* Feedback and improvement suggestions
* 7-day personalized study plan

## Step 6 · Reflection

Without AI, I would have spent several hours researching interview topics, collecting practice questions, and creating a study plan. AI helped me organize the entire preparation process within minutes and provided structured guidance. The prompt-chaining workflow made learning more efficient, focused, and easier to manage.
